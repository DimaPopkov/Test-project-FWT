# О проекте

Проект по созданию и управлению задачами (ToDo листами) пользователя.

## Функционал

В данном проекте представлены функции по:

Управление своими задачами (добавление и удаление задач)
- Легко добавлять и удалять созданные задачи

Пользователь может выбрать любой удобный вариант входа в систему:
- Авторизация (Если у пользователя уже есть аккаунт).
- Регистрация (Для новых пользователей).

## Как запустить локально

### 1. Предварительная настройка

#### В данном проекте используется база данных SQLite, если вам необходимо использовать другую базу данных, воспользуйтесь последующей инструкцией:

В файле `.env ` необходимо настроить подключение к базе данных.
Пример с использованием MySQL:

```plaintext
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=db
DB_USERNAME=root
DB_PASSWORD=
```

Убедитесь, что ваша база данных db развернута на вашем сервере, после этого запустите ниже описанную команду

```bash
# Выполнение миграций для создания таблиц в вашей базе данных
php artisan migrate
```

### 2. Установка зависимостей

Для установки всех зависимостей воспользуйтесь следующими командами:

```bash
# Установка PHP-зависимости
composer install

# Установка JavaScript-зависимостей
npm install
```

### 3. Запуск PHP сервера

Для запуска локального PHP-сервера воспользуйтесь командой:

```bash
php artisan serve
```
### 4. Запуск Front-end сервера

Для запуска сборщика файлов Front-end части сайта выполните в отдельном терминале:

```bash
npm run dev
```

### Требования:
- PHP (8 и выше)
- Composer
- Node.js и nrm
